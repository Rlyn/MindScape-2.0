# 🧠 MindScape — Self-Discovery Psychology Platform

> *Understand your mind. Transform your life.*

MindScape is a fully self-contained, interactive psychology quiz platform built
with **vanilla HTML, CSS, and JavaScript** — no frameworks, no build tools,
no server required. Just open the file in a browser and start exploring.

---

## ✨ Features

- **5 in-depth psychology quizzes** with 35–48 questions each (190+ total)
- **Fully personalized result profiles** with detailed written analysis
- **Visual score breakdowns** — ring charts, dimension bars, and trait grids
- **Concrete step-by-step growth roadmaps** tailored to each result
- **Optional Claude AI enhancement** — paste your Anthropic API key to receive
  a deeply personalized AI-generated analysis on top of your results
- **Zero dependencies** — single `.html` file, works offline

## 🤖 Claude AI Integration (Optional)

Each results page includes an optional AI enhancement feature powered by
**Claude (Anthropic)**. To use it:

1. Get an API key at [console.anthropic.com](https://console.anthropic.com)
2. Complete any quiz and scroll to the **"Enhance with Claude AI"** section
3. Paste your API key and click **Enhance ✨**

Claude will generate a uniquely personalized analysis based on your actual
scores — including tailored strategies, a 30-day challenge, and deeper insight
into your specific profile.

> ⚠️ **Note:** The API key is never stored or transmitted anywhere except
> directly to Anthropic's API. For production deployments, route API calls
> through a backend proxy to protect your key.
